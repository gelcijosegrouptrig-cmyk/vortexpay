💰 Comissões de Parceiros — Histórico de Repasses

ℹ️ Como funciona o pagamento de comissão
✅ Automático via API Mercado Pago
Quando o token MP está configurado, o sistema usa /v1/account/bank_transfers para enviar o PIX direto para a chave do parceiro após cada pagamento confirmado.
⏳ Manual (fallback)
Se a API recusar (sem saldo MP, limite, conta não verificada), o saque fica como pendente_manual — o admin paga manualmente via app Mercado Pago e clica em ✅ Confirmar aqui.
⚠️ A API do Mercado Pago não permite transferir para contas externas sem verificação prévia da conta. Para pagamentos automáticos funcionarem 100%, a conta MP precisa ser verificada e ter saldo suficiente.

💡 Exemplo de Integração Fluxo completo de pagamento
JavaScript — Fluxo completo
const BASE = 'https://web-production-9f54e.up.railway.app';

// 1. Gerar cobrança Pix
const pix = await fetch(`${BASE}/api/pix`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ valor: 50.00 })
}).then(r => r.json());

console.log('Código Pix:', pix.pix_code);
console.log('TX ID:', pix.tx_id);

// 2. Verificar pagamento (polling)
const verificar = async (tx_id) => {
  const status = await fetch(`${BASE}/api/status/${tx_id}`).then(r => r.json());
  if (status.pago) {
    console.log('✅ Pago!');
  } else {
    setTimeout(() => verificar(tx_id), 15000); // retry em 15s
  }
};
verificar(pix.tx_id);
